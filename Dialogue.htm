WorldGrower:Dialogue

Talking to another character is possible by moving next to it and choosing the talk action.



A question or remark is made by the player character and the other character responds.
The relationship with the other character is displayed which indicates how much the other character likes you.
Not all questions will be available all the time.
For example: asking to break up with someone is only possible if you're mates.
Or asking to get cured of poison is only possible if you're poisoned.

A similar dialog is used when another character talks to you:



You have a list of possible responses, just like they do when you talk to them.

For a list of dialogue options, see Dialogue Overview.
